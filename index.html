Создание сайтов.

Меня зовут Андрей, и я занимаюсь созданием сайтов. Основное направление моей деятельности - front-end разработка. Для создания сайтов я использую как «нативные» технологии браузера (HTML, CSS и JS) так и популярные библиотеки и фреймворки (Bootstrap, SCSS, jQuery, React, Vue.js, Node.js).

Кроме front-end разработки я активно изучаю смежные области: back-end, PWA, SEO и продвижение сайтов.

Связаться

CSS Animations

Анимации позволяют добавить на сайт динамичсности, сделать его современным и красивым. Сейчас анимации позволяют сделать практически что угодно, включая 3D. Главное не переобрщить.

Java Script

Java Script добавляет на сайт интерактивность, способность откликаться на различные события и действия, гибко настраивать поведение в зависимости от разных факторов.

HTML5

Стандарт HTML5 добавил много новых возможностей и фишек для работы с изображениями, видео- и аудио-файлами, внутренними системами браузера. Позволяет ещё более тонко настроить и оптимизировать работу сайта.

Адаптивный дизайн.

Очевидно, что сайт должен одинаково хорошо выглядеть на любых устройствах, с которых пользователи будут его видеть. Я достаточно хорошо владею инструментами, позволяющими сделать сайт адаптивным (как с помощю «нативных» технологий, так и с помощю, например, Bootstrap).
